### 一键安装

```haskell
PacletInstall["Waifu","Site"->"http://m.vers.site/"]
```

当然更新和卸载也是一键搞定

```haskell
PacletUpdate["Waifu","Site"->"http://m.vers.site/"]
PacletUninstall["Waifu"]
```
