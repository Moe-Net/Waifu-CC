# Functions

诗云模块


## WaifuTang
### 参数说明

::: tip WaifuTang[length:5,line:4]
- length 类型为 `Integer`, 表示几言诗
- line 类型为 `Integer`, 表示有几句
:::

::: tip WaifuTang[length:5,head]
- length 类型为 `Integer`, 表示几言诗
- head 类型为 `String`, 藏头诗的句首
:::

### 可选项

无

### 返回值

`TableForm`

具体来说是一个 `List`, 每个元素是 `Text`.
