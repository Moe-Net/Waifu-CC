# Waifu:X

## Install

因为 API 快速变动, 所以不适合使用 Paclet Server 分发, 请按以下步骤手动安装

打开以下文件夹
```Mathematica
SystemOpen[FileNameJoin@{$UserBaseDirectory, "Applications"}]
```

接着使用 git 同步以下目录:
```bash
git clone https://github.com/Moe-Net/Waifu-X.git --depth 1 --branch master
```

手动安装的话也是解压到这个目录.
